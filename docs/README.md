### 使用组件库

使用下面两种方式均可安装组件库到项目依赖中。

```sh
npm install widgetui --save  # 使用npm
yarn add widgetui # 使用yarn
```

组件库使用了 less 来开发，所以你还需要安装对应依赖。

```shell script
npm install less less-loader --save-dev
```

### 温馨提示

> 组件库默认使用的 designWidth 为 360px,设计稿需严格按照 360px 来设计，否则会布局异常
